## Panoramica

Questa repository contiene materiale didattico organizzato per argomenti, utile per chi studia Architettura degli Elaboratori. Include:

- **Lezioni teoriche** dettagliate
- **Riassunti** derivati dalle lezioni
- **Domande frequenti agli esami** con risposte
- **Esercizi svolti** per fissare i concetti

> Non rivendico in alcun modo la proprietà delle lezioni teoriche, fornite dall'Università di Padova come materiale didattico a supplemento delle lezioni in aula.

## Struttura della Repository

```
├── Lezioni - Numerate e organizzate/
│   ├── Teoria/                    # 26 lezioni teoriche (PDF)
│   └── Esercitazioni/             # 8 esercitazioni pratiche (PDF)
├── Riassunti/                     # 14 riassunti in Markdown
│   └── PDF/                       # Versioni PDF dei riassunti
└── Test e Domande/                # Domande d'esame ed esercizi
```

## Note

**Materiale incompleto**: Questa repository è un work in progress. Mancano ancora risposte a molte domande aperte d'esame e diversi esercizi per la preparazione. Il contenuto viene aggiornato man mano che procede lo studio.

**Perché questa repository**: Personalmente mi trovo meglio a gestire il materiale in questa maniera piuttosto che dal MEGA comune a tutti gli studenti, che a mio parere risulta spesso confusionario e con file duplicati.

**Obsidian**: Questo progetto è gestito localmente come vault in Obsidian. Alcune funzionalità specifiche di Obsidian (come i link interni `[[...]]`, callout, o altri plugin) potrebbero non funzionare correttamente su altri visualizzatori Markdown.
